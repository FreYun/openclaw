#### 3. 技能内化

**文件**：`skills/xiaohongshu-longform/SKILL.md`

**核心技术点**：
- 小红书长文编辑器必须用 `document.execCommand('insertText')` 输入
- 普通的 `innerText` 或 `value` 方式无效
- 评论框用 `innerText + input event` 即可
- 标题框用 `value + input event` 即可

**完整流程**：
1. 从首页点「发布」→ 新标签页打开创作平台
2. 选「写长文」→「新的创作」
3. 填标题（value + input event）
4. 点编辑器 → execCommand 输入正文
5. 点「一键排版」
6. 点「下一步」
7. 填描述（execCommand）
8. 点「发布」

## 技术细节

### 输入方式对比

| 元素类型 | 输入方式 | 关键代码 |
|---------|---------|---------|
| 普通 input（标题） | `value + input event` | `el.value = 'xxx'; el.dispatchEvent(new Event('input'))` |
| contenteditable（评论） | `innerText + input event` | `el.innerText = 'xxx'; el.dispatchEvent(new Event('input'))` |
| 富文本编辑器（长文） | `execCommand('insertText')` | `document.execCommand('insertText', false, 'xxx')` |

### 注意事项

1. **不要用 URL 直接导航** - 容易被检测，从首页正常点击
2. **页面变化后重新 snapshot** - ref 会失效
3. **长文发布需要封面** - 系统可能自动生成
4. **execCommand 是核心** - 这是搞定富文本编辑器的关键
